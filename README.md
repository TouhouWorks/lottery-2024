## 「幻夢結社」2024 高凉年例祭现场抽奖程序

基于 Vue3 + Vite

部署环境： Vercel

## 环境变量: 

`OTP_SECRET` , 用于 Staff 生成票据 ID 的授权凭证

`JWT_SECRET` , 用于 Staff 登陆后台时生成 JWT 凭据

## 食用方法:

`npx vercel dev`
